# 飞书多维表格模板 - 小红书图文管理

## 表结构

### 主表：内容生产流水线

| 字段名 | 字段类型 | 说明 |
|--------|---------|------|
| 任务 ID | 自动编号 | 唯一标识 |
| 创建时间 | 创建时间 | 自动生成 |
| 选题来源 | 文本 | 原始链接或关键词 |
| 选题类型 | 单选 | 微信文章/关键词/热点 |
| 关键词 | 文本 | 核心关键词 |
| 标题草稿 | 文本 | 生成的标题（多行） |
| 正文内容 | 文本 | 分页内容（多行） |
| 封面文案 | 文本 | 封面大字 |
| 配图说明 | 文本 | 每页配图描述 |
| 图片链接 | 附件/URL | OSS 公网链接 |
| 标签建议 | 文本 | hashtag 列表 |
| 博主名称 | 文本 | 对标博主 |
| 粉丝数 | 数字 | 参考数据 |
| 互动数据 | 文本 | 赞藏评数据 |
| 发布时间建议 | 日期时间 | 建议发布时间 |
| 内容状态 | 单选 | 草稿/待审核/待发布/已发布/已归档 |
| 发布链接 | URL | 发布后的链接 |
| 备注 | 文本 | 其他说明 |

### 附表 1：博主数据表

| 字段名 | 字段类型 |
|--------|---------|
| 博主名称 | 文本 |
| 主页链接 | URL |
| 粉丝数 | 数字 |
| 笔记数 | 数字 |
| 赞藏总数 | 数字 |
| 类目 | 单选 |
| 更新时间 | 日期时间 |

### 附表 2：选题库

| 字段名 | 字段类型 |
|--------|---------|
| 选题内容 | 文本 |
| 选题来源 | 文本 |
| 热度指数 | 数字 |
| 关键词 | 文本 |
| 状态 | 单选（未使用/使用中/已使用） |
| 创建时间 | 日期时间 |

## 自动化配置

### 视图筛选

1. **待处理** - 内容状态 = 草稿
2. **待审核** - 内容状态 = 待审核
3. **待发布** - 内容状态 = 待发布
4. **已发布** - 内容状态 = 已发布

### 机器人配置

1. 创建飞书应用
2. 开通多维表格权限
3. 添加机器人到表格
4. 配置 webhook 通知

---

## OpenClaw 集成

在 OpenClaw 中配置飞书机器人后，AI 可以：
- 读取表格数据
- 写入生成内容
- 更新任务状态
- 发送通知
