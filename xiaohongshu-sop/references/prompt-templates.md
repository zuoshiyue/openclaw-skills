# 小红书提示词模板

## 标题生成

### 公式
```
[关键词] + [情绪词] + [数字] + [结果/收益]
```

### 示例

**知识分享类:**
- "3 天涨粉 1000+，我做对了这 5 件事"
- "保姆级教程！新手必看 7 个技巧"
- "亲测有效！这 3 个方法让我效率翻倍"

**痛点共鸣类:**
- "崩溃！打工人的一天真实写照"
- "谁懂啊！这种痛苦只有 XX 人明白"
- "救命！怎么现在才发现这个神器"

**结果导向类:**
- "坚持 30 天，我收获了什么"
- "从 0 到 1，完整复盘我的 XX 经历"
- "月入 X 万，我是怎么做到的"

## 封面文案

### 原则
- 8-15 字为佳
- 大字突出核心观点
- 颜色对比强烈

### 示例
```
打工人必备神器
3 天涨粉 1000+
保姆级教程
亲测有效
救命神器
```

---

## 分页内容

### 封面页
```
核心观点 + 吸引眼球画面
例：主标题 + 副标题 + 相关配图
```

### 内容页 (1-N)
```
每页一个核心点：
- 标题（简短）
- 正文（1-2 句话）
- 配图建议
```

### 结尾页
```
总结 + 互动引导
例："你学会了吗？评论区告诉我"
```

## 配图建议模板

### 流程图
```
简洁流程图，[数量] 个节点 [排列方式]，
箭头连接，每个节点有图标，简洁现代风格
```

### 对比图
```
左右对比设计，左侧 [问题]，右侧 [解决方案]，
鲜明对比，简洁现代
```

### 数据展示
```
数据图表，柱状图/饼图 + 数字，
[颜色] 色调，科技感，清晰易读
```

### 表情包
```
[情绪] 打工人表情包，[场景]，
夸张漫画风格，高对比度
```

## 内容风格

### 语气
- 亲切自然，像朋友聊天
- 适当使用网络流行语
- 情绪词增强共鸣

### 排版
- 多用 emoji 分隔
- 短段落，易阅读
- 重点内容加粗/标红

### 互动
- 结尾引导评论
- 提问引发讨论
- 收藏 + 关注引导
