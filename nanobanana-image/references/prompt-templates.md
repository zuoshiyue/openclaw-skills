# Nano Banana 提示词模板

## 小红书封面

```
简洁工作流图，左侧输入 [内容]，右侧输出 [结果]，中间 [主题] logo，
[颜色] 色调，扁平化设计，现代简约，无文字
```

## 痛点共鸣

```
[情绪] 打工人表情包，[场景]，夸张漫画风格，高对比度，吸引眼球
```

## 流程图

```
流程图，[数量] 个节点 [排列方式]，箭头连接，每个节点有图标，简洁现代风格
```

## 数据展示

```
数据展示图表，柱状图/饼图 + 数字，[颜色] 色调，科技感，清晰易读
```

## 行动号召

```
行动号召设计，大按钮样式，箭头指向 [方向]，简洁有力，[颜色] 主色调
```

## 示例提示词

### 工作流图
```
简洁工作流图，左侧输入文章链接，右侧输出小红书图文，中间 OpenClaw logo，
蓝白色调，扁平化设计，现代简约
```

### 表情包
```
崩溃打工人表情包，办公桌前抓头，电脑屏幕显示待办事项，夸张漫画风格
```

### 步骤图
```
步骤 1-4 数字图标，圆形背景，水平排列，简洁现代设计
```

## 最佳实践

1. **提示词用英文效果更好**（可选）
2. **避免复杂场景** — 保持画面简洁
3. **颜色指定** — 明确色调（蓝白、蓝绿等）
4. **风格关键词** — 扁平化、现代简约、漫画风格
